Introduction

Qu'est-ce que CSS ? Pourquoi utiliser CSS pour les animations ?

CSS permet de créer des designs web visuellement attrayants avec moins de code et une meilleure performance que JavaScript pour les animations. Cela simplifie le développement et améliore l'expérience utilisateur.

Fondamentaux des animations CSS

Transitions CSS

La syntaxe de base pour une transition CSS se compose de deux parties principales : la propriété que vous souhaitez animer et la durée de cette animation. Par exemple,
`transition: background-color 0.5s;`

- Changer la couleur :

 .color-change:hover {
                background-color: #9EC8B9;}

- Changer la taille :

 .size-change:hover {
                transform: scale(1.1);}

- Rotation :

 .rotate:hover {
                transform: rotate(45deg); 
            }

Animations avec keyframes

Les keyframes en CSS permettent de contrôler les étapes intermédiaires d'une animation, offrant une méthode pour définir précisément comment les propriétés CSS d'un élément changent au fil du temps.

@keyframes moveRightToLeft {
                    0% {
                      transform: translateX(-10);
                    }
                    25% {
                      transform: translateX(40px); /* Se déplace à droite */
                    }
                    75% {
                      transform: translateX(-100px); /* Se déplace à gauche */
                    }
                    100% {
                      transform: translateX(-10); /* Revient au centre */
                    }
                  }

Techniques avancées

1. Contrôle des animations

Animation-delay et animation-direction: Déterminez quand votre animation commence avec animation-delay, et contrôlez son déroulement avec animation-direction.

Animation-fill-mode et animation-play-state: Choisissez comment votre animation applique ses styles au début et à la fin avec animation-fill-mode, et mettez-la en pause ou jouez-la avec animation-play-state.

2. Timing functions

Fonctions linéaires vs. easing: Les animations linéaires changent à une vitesse constante, tandis que les fonctions d'easing créent un mouvement plus naturel en accélérant ou en ralentissant.

Utilisation de cubic-bezier: Avec cubic-bezier, personnalisez précisément la vitesse de votre animation pour un contrôle créatif complet.

3. Performances des animations CSS

Optimisation des animations: Utilisation des propriétés comme transform et opacity pour des animations fluides qui n'exigent pas de recalculs coûteux du layout (reflow) ou du rendu (repaint).
